Inverzno pravilo mešanja       Modifikovana teorija mešanja

$$ E_2 = \frac{E_2^f E_m}{E_mV_f+E_2^fV_m} \ \ \ \ \ \ \ \frac{1}{E_2} = \frac{\frac{V_f}{E_2^f}+\frac{\eta V_m}{E_m}}{V_f+\eta V_m}$$

Model Chamis-a                       Model Halpin-Tsai

$$ E_2 = \frac{E_m}{1-\sqrt{V_f}\left( 1- \frac{E_m}{E_2^f}\right)} \ \ \ \ \ E_2 = \frac{E_m\left( (1+V_f)E_{f2}+V_mE_m \right)}{V_mE_{f2}+(1+V_f)E_m} $$


Pravilo mešanja           Model Hashin-Rosen

$$ G_{12} = \frac{G_{12}^f+G_m}{V_fG_m+V_mG_{12}^f} \ \ \ \ \ \ \ G_{12} = G_m \frac{G_{12}^f(1+V_f)+G_mV_m}{G_{12}^fV_m+G_m(1+V_f)}$$

Model Halpin-Tsai        Model Chamis-a

$$G_{12} = G_m\left(\frac{1+2\xi V_f}{1-\xi V_f}\right) \ \ \ \ \ \ G_{12} = \frac{G_m}{1-\sqrt{V_f}\left( 1- \frac{G_m}{G_{12}^f}\right)} \ \ \ \ \ \ \ \ $$


$$\varepsilon_{ft}^u \lt \varepsilon_{mt}^u \rightarrow F_{1T} = F_{ft}\left( V_f + \frac{E_m}{E_f}V_m \right)$$ $$\varepsilon_{ft}^u \gt \varepsilon_{mt}^u \rightarrow F_{1T} = F_{mt}\left( V_f \frac{E_f}{E_m} + V_m \right)$$

$$ V_f\lt 0.2 \rightarrow F_{1C} = 2V_f\left( \frac{V_fE_fE_m}{3(1-V_f)} \right) ^{1/2} \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ $$$$ V_f\gt 0.2 \rightarrow F_{1C} = \frac{(E_fV_f+E_mV_m)\left(1-V_f^{1/3}\right)\varepsilon_m^u}{\nu_f V_f+\nu_m V_m}$$
